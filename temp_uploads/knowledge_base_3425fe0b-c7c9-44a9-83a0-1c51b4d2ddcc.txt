KEY TERMS GLOSSARY:  
• Machine Learning: A subfield of artificial intelligence that enables systems to learn and improve from experience using data and statistical methods.  
• Supervised Learning: A machine learning approach where models are trained on data that is labeled, meaning the input data is paired with the correct output.  
• Unsupervised Learning: A method where algorithms are given data without explicit instructions on what to do with it, allowing the algorithm to find patterns and relationships on its own.  
• Reinforcement Learning: A type of learning where an agent interacts with an environment, making decisions and receiving feedback through rewards or punishments, thereby learning optimal behaviors.  
• Linear Regression: A statistical technique that models the relationship between a dependent variable and one or more independent variables using a linear equation.  
• Decision Trees: A supervised learning method that uses a tree-like model of decisions and their possible consequences.  
• Neural Networks: Computational models inspired by the human brain, composed of layers of interconnected nodes (neurons) used to recognize patterns and solve complex tasks.  
• Support Vector Machines: A set of supervised learning methods used for classification, regression, and outlier detection that maximizes the separation margin between data points of different classes.
