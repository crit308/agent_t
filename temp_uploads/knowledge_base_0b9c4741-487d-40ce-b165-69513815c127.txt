• Machine Learning: A field within artificial intelligence that employs statistical techniques so that computer systems can "learn" from and make predictions or decisions based on data without explicit programming.  
• Supervised Learning: A learning method in which an algorithm is trained on a dataset that includes input-output pairs (labeled data), enabling the system to predict outcomes for new inputs.  
• Unsupervised Learning: A form of learning where the algorithm works on unlabeled data, seeking to identify underlying patterns or groupings within the dataset.  
• Reinforcement Learning: An approach where an algorithm learns to make sequences of decisions by trial and error, guided by rewards (or penalties) to encourage desirable behaviors.  
• Linear Regression: A simple, widely used algorithm that models the relationship between a dependent variable and one or more independent variables by fitting a linear equation.  
• Decision Trees: A model that uses a tree-like structure to make decisions based on a series of questions, effectively segmenting the data into branches for classification or regression.  
• Neural Networks: Computing systems vaguely inspired by the biological neural networks in brains. They are composed of interconnected nodes (neurons) that process data in layers to capture complex patterns.  
• Support Vector Machines (SVM): A set of supervised learning methods used for classification, regression, and outlier detection, which work by finding the optimal hyperplane that separates data into distinct classes.
